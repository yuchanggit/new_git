
1) what is this?

A code used for skimming. use MakeClass to do. this way can save the whole TTree structure and going to each event in loop

2) how to write the code

according to Eiko's code

https://github.com/syuvivida/xtozh_common/blob/80X_analysis/macro_examples/general/skimTree.h
https://github.com/syuvivida/xtozh_common/blob/80X_analysis/macro_examples/general/skimTree.C

start with make class (see https://www.nevis.columbia.edu/~seligman/root-class/Root-class-02Aug01.pdf)

then modify the code. You can see

to mark where I modify the code from the code generated by MakeClass
